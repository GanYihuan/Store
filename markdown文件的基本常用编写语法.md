# markdown文件的基本常用编写语法

## 1、标题

```
    # 一级标题
    ## 二级标题
    ### 三级标题
    #### 四级标题
    ##### 五级标题
    ###### 六级标题
```

**我是粗体**

*我是斜体*

***我是斜体加粗***

## 换行

```
    md中换行，单纯的回车不能显示成换行效果，可以使用的方法：
    1.在要换行的文本后加两个空格，再回车
    2.回车后，再起一行，也就是回车两次
    3.使用HTML标签<br>
```
